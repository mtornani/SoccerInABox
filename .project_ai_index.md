# AGENT ENTRY POINT - Soccer in a Box

## Project Overview
Soccer in a Box is a "Military Ration" PWA for football academies. It simplifies coaching by providing a fixed, guided session flow and recording minimal feedback (30-second reports).

## Key Files
- `index.html`: Entry point for the PWA HUD.
- `soccer_app_js.js`: core logic (fetching methodology, handling state).
- `assets/methodology.json`: The current session playbooks.
- `soccer_pitch_mobile.html`: High-impact mobile swipeable pitch.

## Deployment

## AI Instructions
When starting work here, always check `assets/methodology.json` to understand the current tactical curriculum being distributed. Logic is strictly "Military Ration" - avoid adding complex features that dilute the essential nature of the toolkit.
